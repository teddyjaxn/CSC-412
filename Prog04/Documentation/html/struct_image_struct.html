Detailed Description

This is the data type to store all relevant information about an image. After some thought, I have decided to store the 1D and 2D rasters as void* rather than having separate unsigned char and float pointers

void* raster

Pointer to the image data, cast to a void* pointer. To access the data, you would have to cast the pointer to the proper type, e.g.
